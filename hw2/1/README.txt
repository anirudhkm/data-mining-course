This is a README for the program iris_analysis.py

1. The programs takes the iris data as a system argument and also the iris headers as a system argument.

2. With the help of dataframe constructed from this data, we calculate the mean and standard deviation for all the features.

3. The same step is repeated for all the flower types.

4. Next the boxplots are plotted for the data for each feature based on the flower type.

5. Finally the figures are saved to current working directory.
